# Arquitectura MVVM

## Responsabilidades por Capa
- **Screens** - Composables de pantallas completas
- **Components** - UI reutilizable y específica
- **ViewModels** - Estado de UI y lógica de presentación
- **Navigation** - Flujo entre pantallas
